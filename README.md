# myszkowski-cipher-project
# 🔐 Myszkowski Cipher with Custom Hash

## 📌 Overview

This project implements the **Myszkowski Transposition Cipher** along with a **custom hashing function** in Python.

It includes:

* Encryption and Decryption of text
* Custom Hash Generation
* Full round-trip verification (Encrypt → Hash → Decrypt)

---

## 🔑 Myszkowski Cipher (Theory)

The Myszkowski Cipher is a **transposition cipher**, meaning it rearranges the positions of characters instead of changing them.

### 🔍 How it Works

1. Write plaintext row-wise into a matrix based on key length
2. Assign ranks to key letters:

   * Alphabetically sorted
   * Same letters → same rank
3. Read columns based on rules:

### 📏 Rules

* 🔹 Unique key letters → read column-wise (top to bottom)
* 🔹 Repeated key letters → read row-wise across those columns

## 🔐 Encryption Process

Plaintext: `HELLO WORLD`
Key: `BANANA`

### Step 1: Remove spaces & uppercase

```
HELLOWORLD
```

### Step 2: Fill matrix (row-wise)

```
H  E  L  L  O  W
O  R  L  D  _  _
```

(_ = padding)

### Step 3: Read using Myszkowski rules

Ciphertext:

```
HLOELWRLOD
```

---

## 🔓 Decryption Process

1. Recreate matrix structure
2. Fill characters using same key order
3. Read row-wise

Output:

```
HELLOWORLD
```

## 🔢 Custom Hash Function

We implemented a custom polynomial-style hash:

### Formula:

```
H = (H * 37 + ASCII(char)) mod (10^9 + 7)
```

### Why this hash?

* Efficient (O(n))
* Good distribution
* Avoids trivial collisions
* Different from standard MD5/SHA (assignment requirement)
